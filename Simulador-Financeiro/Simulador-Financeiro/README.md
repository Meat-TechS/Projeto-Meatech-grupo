# Calculadora Financeira - Câmara Fria

Este projeto é uma calculadora financeira simples em HTML e JavaScript que simula o impacto financeiro de perdas em câmaras frias devido a problemas de temperatura.

# Tecnologias utilizadas
- HTML5
- JavaScript

# Interface do projeto
AQUI VAI A FOTO DA CALCULADORA, ESPERANDO O CSS
# A lógica principal segue esses passos:
- Calcula o total de kg armazenados
- Calcula a perda
- Calcula o prejuízo
- Simula economia com sistema

# Objetivo
Demonstrar de forma simples e visual como perdas por temperatura impactam financeiramente um negócio e como um sistema de monitoramento pode gerar economia.
